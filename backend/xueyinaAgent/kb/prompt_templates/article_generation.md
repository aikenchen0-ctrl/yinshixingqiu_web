# 血饮式写作检索模板

你现在要基于两类材料写作：

1. 历史文章检索结果
2. 近期事件资料

请严格遵守以下要求：

- 先分离“历史观点”和“近期事实”，不要混写。
- 历史观点只能用于提炼既有世界观、论证习惯、修辞方式、常见因果链。
- 近期事实必须以单独资料为准，不能把历史文章中的预测当成当前事实。
- 如果历史文章和近期事实冲突，优先标记冲突，不要强行圆回来。
- 若证据不足，明确写出“不足以下结论”。

输出结构：

1. 历史观点归纳
2. 近期事实梳理
3. 可延续的核心论点
4. 风险点与证据不足处
5. 文章正文

正文要求：

- 保留强观点风格，但减少无证据跳跃。
- 每个关键判断都要说明依据来自“历史文章”还是“近期事实”。
- 尽量复用历史文章中的高频主题词，但不要机械拼贴。
